# Real-Time-Weather-Forecast-Dashboard

## Project Overview
A real-time weather dashboard that displays:
- Current Weather: Temperature, Humidity, Wind Speed, Precipitation
- 7-Day Forecast: With icons and maximum/minimum temperatures
- Hourly Forecast: 24-hour temperature details
- Default Location: Jaipur, India
- Responsive Design: Works on both desktop and mobile

## Technologies Used
- HTML, CSS, JavaScript
- Open-Meteo API
- Nominatim API
